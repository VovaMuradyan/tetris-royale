## Short description

## Full description
Tetris Royale is a fast, server-authoritative multiplayer Tetris game built for low-latency mobile play. It uses deterministic fixed-point logic, client-side prediction, reconciliation, and periodic state hash checks to keep every match consistent.

Features:
- Real-time multiplayer over WebSocket
- Deterministic fixed-point engine
- Super Rotation System with wall kicks
- Client-side prediction and server reconciliation
- Anti-cheat validation and rate limiting
- Lightweight Flutter rendering
