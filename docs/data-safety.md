# Data Safety Draft

Use this as the factual basis for Play Console.

## Collected data
- App activity: gameplay interactions, taps, match actions, session events
- Device or other IDs: Firebase Analytics identifiers
- App info and performance: crash logs and stability diagnostics
- Network metadata: IP address and connection timestamps are processed by the backend to operate matches and prevent abuse

## Shared data
- Shared with service providers only for operation and analytics

## Purpose
- App functionality
- Analytics
- Security and fraud prevention
- Diagnostics

## Retention
- Retained only as needed for matchmaking, debugging, abuse prevention, and legal compliance

## Notes
- No account creation
- No ads
- No payments
